Overview

Running in parallel with our first Advanced cohort, this second beginners’ edition took the proven structure of Edition 1 and tightened the pacing and examples. The destination was the same and deliberately ambitious: every student writing, running and understanding real code by the end of term.

What students explored

  • A clear model of how code runs before diving into syntax.
  • Core programming — variables, conditionals, loops and functions.
  • Debugging — turning errors from scary to solvable.
  • Version control — saving, tracking and not fearing change.
  • Hands-on builds — frequent small projects to lock in each new idea.
  • Confidence first — a pace that makes sure nobody is left behind.

Skills developed

  • Transferable programming fundamentals.
  • Methodical problem-solving and debugging.
  • The confidence to keep building independently after the course.
